🚀 TRY THIS FIRST!

This guide will help you run your first analysis in less than 5 minutes.

═══════════════════════════════════════════════════════════════════════════════
📋 STEP 1: INSTALLATION
═══════════════════════════════════════════════════════════════════════════════

Run the automatic installation (creates virtual environment and installs dependencies):

    ./analyse_conversations_merged.py --install

═══════════════════════════════════════════════════════════════════════════════
🔑 STEP 2: CONFIGURE API KEY
═══════════════════════════════════════════════════════════════════════════════

Set your Mistral API key:

    export MISTRAL_API_KEY='your_api_key_here'

💡 To get a Mistral API key: https://console.mistral.ai/

═══════════════════════════════════════════════════════════════════════════════
✨ STEP 3: RUN THE EXAMPLE
═══════════════════════════════════════════════════════════════════════════════

Run the security analysis on the example data:

./analyse_conversations_merged.py --exec --recursive --aiall \
  --fichier ./data_example/ \
  --prompt-file example_security_prompt \
  --target-logs ./logs_example/ \
  --target-results ./results_example/ \
  --format markdown

This command will:
✅ Analyze all conversations in data_example/
✅ Apply the security analysis prompt
✅ Generate a detailed Markdown report
✅ Save logs in logs_example/
✅ Save results in results_example/

═══════════════════════════════════════════════════════════════════════════════
📊 STEP 4: VIEW THE RESULTS
═══════════════════════════════════════════════════════════════════════════════

Check the generated report:

    cat results_example/results_example_security_prompt_*.md

Or open it in your browser/editor to see the formatted Markdown.

═══════════════════════════════════════════════════════════════════════════════
🎯 NEXT STEPS: CREATE YOUR OWN ANALYSIS
═══════════════════════════════════════════════════════════════════════════════

1. EXPORT YOUR CONVERSATIONS:
   - ChatGPT: Settings → Data Controls → Export data
   - Claude: Settings → Export conversations
   - LeChat/Mistral: Export your conversations as JSON

2. CREATE YOUR DATA DIRECTORY:
   
   mkdir -p ./data
   # Copy your exported JSON files here

3. CREATE YOUR CUSTOM PROMPT:
   
   nano prompts/prompt_my_analysis.txt
   
   Example content:
   ---SYSTEM---
   You are an expert in [YOUR DOMAIN].
   
   ---USER---
   Analyze this conversation about {TITLE}.
   
   Conversation:
   {CONVERSATION_TEXT}
   
   Provide: [YOUR INSTRUCTIONS]

4. RUN YOUR ANALYSIS:

./analyse_conversations_merged.py --exec \
  --aiall --recursive \
  --fichier ./data/ \
  --prompt-file my_analysis \
  --target-logs ./logs/ \
  --target-results ./results/ \
  --format markdown
